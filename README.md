Not very practical AI-made wheels, a desktop GUI version of Git.

## Dependencies

- **Framework**: Wails v2 (Go backend + WebView frontend)
- **Frontend**: React + TypeScript + Vite
- **Frontend Libraries**: reactflow, dagre
- **Backend**: Go (git CLI)
- **Target Platform**: Windows

## Usage

Install [Git](https://git-scm.com/downloads) on your computer first, until Git is able to run in your terminal.

Then run the app and TRY IT YOUSELF.
